# Mini timer

React로 개발한 타이머 토이 프로젝트  


<br>

## 프로젝트 설명
<br>

- React로 개발한 SPA 앱  
- 상태 관리를 위해 Redux 라이브러리 활용. 
- 버튼, 박스, 시간등의 위젯은 Bootstrap 라이브러리에서 import

<br>

**Start** : 타이머 시작  
**Pause** : 일시정지  
**Reset** : 초기화  
**Round** : 기록  

<br>

### React로 개발한 이유
<br>

- 사용자와 상호작용 구현이 용이
- state에 따라 다르게 렌더링되는 컴포넌트
- 가장 널리 쓰이는 프론트엔드 라이브러리 사용 경험 축적


<br>

### 구현 기능
<br>

- Start, Pause, Reset 등 타이머 기초 기능
- Round로 기록 목록에 추가
- Round가 목록 높이보다 더 많이 쌓일시 스크롤
